Probability: The probability of an event is a number that represents the likelihood of the event occurring. Probabilities are always between 0 and 1, with 0 meaning an impossible event and 1 meaning a certain event.

# Random Variables
A random variable is a variable whose value is uncertain. Probability theory is used to describe the behavior of random variables and make predictions about their values.

# Conditional Probability
Conditional probability is the probability of an event happening given that another event has already happened. This concept is used to make predictions based on partial information.

## Bayes' Theorem
Bayes' theorem is a formula that relates the probabilities of two events, and provides a way to update probabilities as new information becomes available.

## Independence
Independence: Two events are independent if the occurrence of one event does not affect the likelihood of the other event happening. Understanding independence is important for modeling and making predictions in many fields.

## Expectation
Expectation: The expectation of a random variable is a value that represents the average or expected value of the variable. This concept is used to make predictions about the behavior of random variables and to make decisions based on uncertain information.

# Standard Deviation
Standard deviation: The standard deviation of a random variable is a measure of the spread or variability of the variable. Understanding the standard deviation is important for understanding the behavior of random variables and making predictions about their values.
